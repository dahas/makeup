makeup

A package for proper web development.

What is it?

makeup is an easy to use PHP framework and a compilation of popular frontend libraries. It is a package that includes almost everything you need to build awesome web applications from the ground up.

Read more ...

Requirements

In order to get makeup to work, you need PHP 5.4 or above, a MySQL database and a local webserver. If you have installed Xampp or something similar you are already set up. Also you need Git and Composer available in your environment.

Read more ...

Download

Download makeup from GitHub. Or make a clone: 

$ git clone https://github.com/dahas/makeup.git

Get started

  1. Download and unpack the source files into a folder within the document root of your webserver.
  2. Install dependent libraries
    $ composer install
  3. Create file index.php in root. Paste following code into it: 
    %CODE%
  4. Launch app in your web browser.
